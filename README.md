# VibeSphere
**VibeSphere** is a smart, mood-based music recommendation web application built
with Python and Streamlit. It suggests the perfect tracks based on your current
emotional state and preferred language/genre.
Whether you are feeling heartbroken, energetic, or just want to chill, VibeSphere curates
a playlist for you with direct links to listen.
## Features
* **Mood Detection:** Choose a mood from a list or type how you feel (e.g., *"I had a
rough day at work"*), and the built-in Keyword Analyzer will detect your vibe.

## Tech Stack
* **Language:** Python 3.x
* **Framework:** [Streamlit](https://streamlit.io/)
* **Logic:** Custom Keyword Matching Algorithm (NLP-lite) for mood analysis.
